What makes this different

Discovery Phase (Fast AI)

Reads every document. Extracts every fact. Links each fact to an exact quote from the source.

Rule: No inference. Only explicit statements become facts. "SAP has 500 users" = fact. "SAP is probably critical" = not a fact.

Reasoning Phase (Smart AI)

Analyzes facts through M&A lens. Generates risks and work items. Calculates cost implications.

Rule: Every risk must cite specific facts. No conclusion without evidence. The "so what" is required.
