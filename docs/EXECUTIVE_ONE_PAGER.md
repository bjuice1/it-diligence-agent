# IT Due Diligence Agent
## Executive Summary

---

### What Is It?

An AI-powered tool that **assists with the first pass of IT due diligence**, helping the team produce consistent, evidence-backed analysis more efficiently.

```
┌──────────────┐      ┌──────────────┐      ┌──────────────┐
│  IT Docs     │      │  AI-Assisted │      │  Structured  │
│  from VDR    │ ───► │  Analysis    │ ───► │  Outputs     │
└──────────────┘      └──────────────┘      └──────────────┘
                             │
                      Team reviews &
                      adds judgment
```

---

### What It Produces

| Output | Description |
|--------|-------------|
| **Facts Inventory** | Extracted information with source quotes |
| **Risk Register** | Identified risks with severity and mitigation |
| **VDR Requests** | Follow-up questions generated from gaps |
| **Coverage Score** | Grade (A-F) showing documentation completeness |
| **Cost Estimates** | Integration costs by phase, aggregated |

---

### What It Does Well

- Applies same checklist consistently across all deals
- Catches items that might be missed under time pressure
- Traces every finding back to source documents
- Generates structured follow-up questions automatically
- Aggregates cost estimates by phase and owner

### What Still Needs Human Judgment

- Deal-specific context and strategy implications
- Severity calibration for unusual situations
- Industry-specific nuances not yet encoded
- Final validation before IC distribution

---

### How Team Members Improve It

| Action | Effort | Impact |
|--------|--------|--------|

| Flag missed pattern | 10 min | All future deals evaluate it |
| Calibrate estimate | 15 min | More accurate budgeting |

**The system captures expertise and applies it consistently.**
