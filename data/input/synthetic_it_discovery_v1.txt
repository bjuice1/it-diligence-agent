SECTION 4: CYBERSECURITY POSTURE
================================================================================

## Security Organization
- CISO: Position filled in 2023 (previously reporting to CIO)
- Security Team: 5 FTEs (CISO + 4 analysts/engineers)
- Security Operations: Outsourced to managed security provider (Secureworks)

## Security Tools & Controls
- SIEM: Splunk Enterprise (deployed 2022)
- EDR: CrowdStrike Falcon
- Email Security: Proofpoint
- Vulnerability Management: Tenable.io
- PAM: CyberArk (implementation in progress, 60% complete)
- DLP: Microsoft Purview (pilot phase)

## Identity & Access Management
- Active Directory: Windows Server 2019, single forest
- Azure AD: Hybrid configuration for M365
- SSO: Okta (deployed 2023, 85% of SaaS apps integrated)
- MFA: Okta Verify for remote access and SaaS
  - MFA NOT enforced for on-premises applications
  - Legacy VPN (Cisco AnyConnect) uses password only

## Recent Security Incidents
- March 2024: Phishing attack resulted in compromised credentials
  - 3 executive accounts affected
  - Contained within 4 hours, no data exfiltration confirmed
  - Led to accelerated MFA deployment

- June 2023: Ransomware attempt blocked by CrowdStrike
  - Entry point: USB device at manufacturing site
  - No systems encrypted, investigation completed

## Compliance & Audits
- SOC 2 Type II: Not certified, assessment planned for 2025
- PCI-DSS: Not applicable (no card processing)
- Last External Penetration Test: Q2 2024
  - 3 critical findings (all remediated)
  - 12 high findings (8 remediated, 4 in progress)

## Known Security Gaps
- No formal security awareness training program
- Patch management SLA not consistently met (target: 30 days for critical)
- Third-party risk management process is informal
- No dedicated OT/ICS security controls for manufacturing systems
